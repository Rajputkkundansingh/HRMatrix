Smart HR Contact Manager | Spring Boot, MySQL (Aug 2024)
The Smart HR Contact Manager is a web-based application designed to streamline the management of HR-related contacts with secure access, user-friendly features, and efficient data handling. This platform offers a modern solution for HR professionals to store, manage, and export contact information, enhancing organizational efficiency.
Key Features:
1. Secure User Authentication:
Integrated OAuth-based login using Spring Security, allowing users to log in via Google and GitHub, ensuring secure and hassle-free access.
2. Personalized Dashboards:
Each user can access a personalized dashboard for managing their HR contacts efficiently.
3. Responsive Frontend:
Developed with Thymeleaf, TailwindCSS, and JavaScript to ensure a seamless and responsive user experience across devices.
4. Robust Backend:
Built using Spring Boot to handle HTTP requests, manage user sessions, and implement core business logic.
5. Database Management:
Designed and optimized MySQL databases to store HR contacts, user data, and associated files, ensuring data integrity and quick retrieval.
6. File Storage:
Integrated Cloudinary SDK to facilitate storing and managing user-uploaded profile pictures and documents.
7. Export Functionality:
Users can easily export contact information for offline use and reporting purposes.
8. Automated Email Notifications:
Implemented Java-based email services to send automated notifications, keeping users informed of updates and actions.
